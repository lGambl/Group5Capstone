Context: User has loaded into the source explorer
Primary Flow:
1.  The user selects a source to view
2.  The user is directed to a page for viewing where the source and its notes are displayed

Alternative Flow(s):
User closes/goes back:
    3a.  The user selects the back button or closes the window
    4a.  The user is returned back to the source explorer
User's source fails loading:
    2b.  A prompt appears informing the user that the source is unavailable.
    3b.  The user is returned back to the source explorer
